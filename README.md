# nuxt-electron-example

## Build Setup

```bash
# install dependencies
$ yarn install

# serve with hot reload
$ yarn dev

# build for production
$ yarn build # for building environment
$ yarn build:all
$ yarn build:win
$ yarn build:mac
$ yarn build:linux
```
